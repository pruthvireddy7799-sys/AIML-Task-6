# Iris Dataset - Machine Learning Classification

## 📌 Objective
To build a machine learning model using Logistic Regression to classify Iris flower species based on their features.

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Dataset
Iris Dataset (Iris.csv)

## 🚀 Steps Performed
1. Imported required libraries
2. Loaded dataset using Pandas
3. Cleaned data and removed unnecessary columns
4. Encoded target variable (Species)
5. Split dataset into training and testing sets
6. Standardized features using StandardScaler
7. Trained Logistic Regression model
8. Made predictions on test data
9. Evaluated model using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
10. Visualized data using pairplot

## 📈 Results
- Model achieved high accuracy due to clear separation between classes
- Confusion matrix shows correct classification of most samples
- Classification report indicates strong precision and recall

## 🔍 Observations
- Iris dataset is well-structured and easy to classify
- Logistic Regression performs well for multi-class classification
- Data visualization helps understand feature relationships

## ✅ Conclusion
The Logistic Regression model successfully classified Iris species with high accuracy. This demonstrates the effectiveness of simple machine learning models on structured datasets.
